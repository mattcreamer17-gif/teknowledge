# TeKnowledge landing page

Bespoke prospect-facing page for TeKnowledge, a 4,000-plus-person Microsoft-pinned services firm whose AI-First rebrand (April 2025) needs a portable methodology layer underneath a growing adoption practice. The page frames a peer-level conversation about the ICM methodology, maps Eduba's 60/30/10 orchestration frame onto their Qatar-scale Copilot work, and asks for a 30-minute call with Matt Creamer.

Built with TeKnowledge brand tokens: the deep indigo to electric violet gradient hero (`#16064A` to `#4F45ED` to `#9333EA`), coral-orange (`#FF835D`) for sparing accents and CTAs, and a clean white surface below the hero in the style of their own insights pages. Inter is loaded as the fallback because All Around Gothic and Overused Grotesk are proprietary. No JavaScript. Single stylesheet. Renders on 375px.

Intended URL path: /for/teknowledge
